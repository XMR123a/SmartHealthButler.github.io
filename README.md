# 智能健康管家APP - 展示项目

> 以AI大模型+多维度健康数据为核心，打造集健康监测、风险预警、个性化干预、在线医疗、慢病管理于一体的一站式智能健康管理平台

## 项目简介

智能健康管家APP是面向全年龄段用户的轻量化、精准化、全天候主动健康服务平台。我们通过AI技术实现"数据统一分析+饮食识别+运动指导+语音陪伴+AI问诊"全链路自动化闭环，帮助用户从被动就医转向主动健康管理。

## 项目特色

### 🤖 AI智能问诊
- 智能分诊与多轮引导式提问
- 症状分析与风险等级评估
- 科室推荐与就医建议
- 用药安全自动核对

### 🍎 AI饮食管理
- 食物数据库与营养分析
- 热量与营养成分计算
- 支持多人家庭饮食记录
- 个性化饮食建议

### 📊 AI健康分析
- 健康问卷与数据采集
- 睡眠结构分析
- 活动能量可视化
- AI健康日报生成

### 🎤 AI语音陪伴
- 7×24小时语音对话
- 健康提醒与用药通知
- 呼吸训练与助眠陪伴
- 语音合成与识别

## 快速开始

### 环境要求
- 现代浏览器（Chrome、Firefox、Safari、Edge）
- 无需特殊环境，纯HTML/CSS/JavaScript实现

### 运行方式

#### 方式一：直接打开
直接在浏览器中打开 `index.html` 文件

#### 方式二：本地服务器（推荐）

使用Python：
```bash
python -m http.server 8000
```

使用Node.js：
```bash
npx http-server -p 8000
```

然后访问：`http://localhost:8000`
